# 💰 SmartSpend

An AI-based Personal Expense Tracker web app built with Django (Python).

## 🚀 Features
- User Login & Logout
- Add & Delete Expenses
- Category-wise tracking (Food, Travel, Education, Bills, Entertainment)
- Dashboard with Bar Chart (Chart.js)
- Total expense summary

## 🛠️ Tech Stack
- Backend: Django (Python)
- Frontend: HTML, CSS
- Database: SQLite
- Charts: Chart.js
